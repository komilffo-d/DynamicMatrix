This is a software with the ability to perform various actions on matrices.

## Features
- Matrix addition (Pair operation)
- Matrix Subtraction (Pair operation)
- Matrix multiplication (Pair operation)
- Matrix division (Pair operation)
- Multiplying a matrix by a number (Solo operation)
- Finding the determinant of the matrix (Solo operation)
- Matrix transposition (Solo operation)
- Inverse matrix (Solo operation)
- Export confugiration's matrices to TXT File
- Import configuration's matrices from TXT File
- Export result to Word file
- Export result to TXT File
- Loading data into the Database during one of the operations with data entry with input ,output data and type operation

## Tech Stack

**Client:** .NET 7, WinForms, DLL C#, COM Word Object, EF Core, MS SQL

## Related

As part of its implementation, the project partially uses the MathNet library,

[MathNet Nuget Package](https://www.nuget.org/packages/MathNet.Numerics/)
